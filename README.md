# Aetherpaloo Community Templates

A privacy-first library of reusable chapter-opening designs for [Aetherpaloo](https://aetherpaloo-web.clapp0528.chatgpt.site).

## The rule that matters

A template contains **design settings only**. It must never contain manuscript text, chapter titles, book metadata, cover art, or embedded images.

## Use a template

1. Download an `.aethertemplate.json` file from [templates](templates).
2. In Aetherpaloo, open **Design → Style Templates**.
3. Choose **Import Community Template**.
4. Review it on the editable first-chapter sample.
5. Apply the design across the book when it looks right.

Unique chapter titles and chapter-specific images are never copied across the manuscript.

## Share a template

1. Create or adjust a Custom Template in Aetherpaloo.
2. Choose **Share with Community**.
3. Confirm the privacy notice.
4. Attach the downloaded template file and a preview screenshot to a submission issue.
5. A maintainer validates the file before publishing it.

See [CONTRIBUTING.md](CONTRIBUTING.md) and the [template schema](schema/template.schema.json).

## Repository layout

- `index.json` — machine-readable published-template catalog
- `templates/` — reviewed template files
- `schema/` — format validation
- `.github/ISSUE_TEMPLATE/` — guided submissions
